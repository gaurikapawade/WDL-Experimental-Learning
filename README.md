## Experimental Learning
This project is a **Student Course Feedback Form** developed for the Web Designing Lab.
The form collects structured feedback from students regarding courses, instructors, and overall learning experience.

### Input Types Used

* Text
* Email
* Number
* Radio buttons
* Checkboxes
* Range sliders
* Textareas
* Dropdown lists

### Structure & Implementation

* Organized content using `<fieldset>` and `<legend>`
* Implemented basic validation using `required`

### Features

* Structured multi-section feedback form
* Required field validation
* Multiple input types for accurate data collection
* Clean and responsive UI design
* Separate success page after submission
* External CSS for maintainability
